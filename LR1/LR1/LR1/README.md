# Лабораторная работа №1

## Обработка строки символов

### Описание программы

Программа выполняет обработку строки символов в три этапа:

1. **Фильтрация** — удаляет все символы, кроме букв и цифр (включая символы кириллицы)
2. **Переворот** — инвертирует порядок символов в строке
3. **Удвоение** — удваивает каждый третий символ (символы с индексами 2, 5, 8, 11, ...)

### Пример работы

**Входная строка:** `Hello123 World!`

1. После фильтрации: `Hello123World`
2. После переворота: `dlroW321olleH`
3. После удвоения: `dlroW3321olleH` (символы с индексами 2, 5, 8 удвоены)

### Варианты реализации

Программа предоставляет три варианта работы со строками:

#### Вариант 1: Строка как массив
- Использует массив символов фиксированного размера
- Ввод данных с клавиатуры
- Вывод результата на экран

#### Вариант 2: Строка C-style с указателями
- Использует динамическое выделение памяти (`new char[]`)
- Работа с указателями при фильтрации
- Ввод данных с клавиатуры
- Вывод результата на экран

#### Вариант 3: Работа с файлами
- Чтение строки из файла `input.txt`
- Запись результата в файл `output.txt`
- Автоматический поиск файла `input.txt` в нескольких возможных расположениях
